#タスク管理アプリ

![task.gif](https://qiita-image-store.s3.amazonaws.com/0/140284/90f36148-7219-7807-9dd7-44e7e4a827bc.gif)

<br>

タスク管理アプリをscaffoldで作成します。

- アプリ名 rails_task
- モデル名 task
- scaffoldで作成
- 日本語化(画面のラベルを日本語に変更)

<br>

## データベース定義

カラム名とデータ型は以下の通りです。

| カラム名|データ型| 
|:-----------|:------------|
| タスク名: name       |       string |
| カテゴリ: category     |      string |   
| 期限: due_date       |        date |    
| メモ: memo         |          text |
